腾讯2020校园招聘-后台：逆序对

发表于2023-03-31|更新于2023-04-01|笔试真题腾讯
|字数总计:1.1k|阅读时长:5分钟|阅读量:

题目链接：https://www.nowcoder.com/questionTerminal/8fe007e54fc04b5e82089aaa71ba3553

题目描述

作为程序员的小Q，他的数列和其他人的不太一样，他有 2n2^n2n 个数。

老板问了小Q一共 mmm 次，每次给出一个整数 qi (1≤i≤m)q_i \ \left( 1 \leq i \leq m \right)qi​ (1≤i≤m)，要求小Q把这些数每 2qi2^{q_i}2qi​ 分为一组，然后把每组进行翻转，小Q想知道每次操作后整个序列中的逆序对个数是多少呢？

例如：

对于序列 1 3 4 2，逆序对有 (4,2)\left( 4, 2 \right)(4,2)、(3,2)\left( 3, 2 \right)(3,2)，总数量为 222。

翻转之后为 2 4 3 1，逆序对有 (2,1)\left( 2, 1 \right)(2,1)、(4,3)\left( 4, 3 \right)(4,3)、(4,1)\left( 4, 1 \right)(4,1)、(3,1)\left( 3, 1 \right)(3,1)，总数量为 444。

输入描述

第一行一个数 n (0≤n≤20)n \ \left( 0 \leq n \leq 20 \right)n (0≤n≤20)

第二行 2n2^n2n 个数，表示初始的序列 ai (1≤ai≤109)a_i \ \left( 1 \leq a_i \leq 10^9 \right)ai​ (1≤ai​≤109)

第三行一个数 m (1≤m≤106)m \ \left( 1 \leq m \leq 10^6 \right)m (1≤m≤106)

第四行 mmm 个数，表示 qi (0≤qi≤n)q_i \ \left( 0 \leq q_i \leq n \right)qi​ (0≤qi​≤n)

输出描述

mmm 行，每行一个数表示答案。

示例

输入

2
2 1 4 3
4
1 2 0 2

输出

0
6
6
0

说明

初始序列 2 1 4 3

2q1=22^{q_1} = 22q1​=2 -> 第一次：1 2 3 4 -> 逆序对数为 000

2q2=42^{q_2} = 42q2​=4 -> 第二次：4 3 2 1 -> 逆序对数为 666

2q3=12^{q_3} = 12q3​=1 -> 第三次：4 3 2 1 -> 逆序对数为 666

2q4=42^{q_4} = 42q4​=4 -> 第四次：1 2 3 4 -> 逆序对数为 000

题解

假设原数组为 {8,1,2,5,7,4,3,6}\left\{ 8, 1, 2, 5, 7, 4, 3, 6 \right\}{8,1,2,5,7,4,3,6}，求原数组的逆序对数如下图所示。

将原数组 {8,1,2,5,7,4,3,6}\left\{ 8, 1, 2, 5, 7, 4, 3, 6 \right\}{8,1,2,5,7,4,3,6} 每 222^222 个为一组进行翻转得到 {5,2,1,8,6,3,4,7}\left\{ 5, 2, 1, 8, 6, 3, 4, 7 \right\}{5,2,1,8,6,3,4,7}，求翻转后数组的逆序对数如下图所示。

从上图我们可以观察到，对数组每 222^222 个进行翻转时，reorder[1] 和 order[1] 进行了互换，reorder[2] 和 order[2] 也进行了互换。

所以，对数组每 2i2^i2i 个进行翻转时，我们可以先把 1∼i1 \sim i1∼i 的 reorder 和 order 数组元素互换，然后通过计算 reorder 数组的累加和来求得数组的逆序对数。

#include <iostream>
#include <cstdio>
#include <algorithm>

using namespace std;

typedef long long ll;

const int N = (1 << 20) + 10;

int arr[N], rearr[N], tmp[N];

ll order[30], reorder[30];

// 将[l1,r1]和[l2,r2]两个有序区间合并为一个有序区间
void Merge(int nums[], int l1, int r1, int l2, int r2, ll inversions[], int index)
{
    int i = l1, j = l2, k = l1;
    while (i <= r1 && j <= r2)
    {
        if (nums[i] > nums[j])
        {
            inversions[index] += r1 - i + 1;
            tmp[k++] = nums[j++];
        }
        else
        {
            tmp[k++] = nums[i++];
        }
    }

    while (i <= r1) tmp[k++] = nums[i++];

    while (j <= r2) tmp[k++] = nums[j++];

    for (int i = l1; i <= r2; i++)
    {
        nums[i] = tmp[i];
    }
}

// 求数组nums的逆序对数量inversions
void MergeSort(int nums[], int left, int right, ll inversions[], int index)
{
    // 若当前区间只有一个数或没有数，则直接return
    if (left >= right) return;

    // 将[left,right]划分为[left,mid]和[mid+1,right]两个子问题
    int mid = (left + right) >> 1;
    MergeSort(nums, left, mid, inversions, index - 1);
    MergeSort(nums, mid + 1, right, inversions, index - 1);

    // 将[left,mid]和[mid+1,right]两个有序区间合并为一个有序区间
    Merge(nums, left, mid, mid + 1, right, inversions, index);
}

int main()
{
    int n;
    scanf("%d", &n);

    int tot = 1 << n;

    for (int i = 0; i < tot; i++)
    {
        scanf("%d", &arr[i]); // 原数组arr
        rearr[tot - 1 - i] = arr[i]; // 反转后的数组rearr
    }

    MergeSort(arr, 0, tot - 1, reorder, n); // 求原数组arr的逆序对数量

    MergeSort(rearr, 0, tot - 1, order, n); // 求原数组arr的顺序对数量等价于求反转后数组rearr的逆序对数量

    int m;
    scanf("%d", &m);

    while (m--)
    {
        int q;
        scanf("%d", &q);

        for (int i = 1; i <= q; i++)
        {
            swap(order[i], reorder[i]);
        }

        ll res = 0;
        for (int i = 1; i <= n; i++)
        {
            res += reorder[i];
        }
        printf("%lld\n", res);
    }

    return 0;
}
