腾讯2020校园招聘-后台：视野争夺

发表于2023-04-17|更新于2023-04-17|笔试真题腾讯
|字数总计:582|阅读时长:2分钟|阅读量:

题目链接：https://www.nowcoder.com/questionTerminal/61e1e66e39f348cdb6495de91ac36a41

题目描述

小Q在进行一场竞技游戏，这场游戏的胜负关键就在于能否能争夺一条长度为 LLL 的河道，即可以看作是 [0,L]\left[ 0, L \right][0,L] 的一条数轴。

这款竞技游戏当中有 nnn 个可以提供视野的道具-真视守卫，第 iii 个真视守卫能够覆盖区间 [xi,yi]\left[ x_i, y_i \right][xi​,yi​]。现在小Q想知道至少用几个真视守卫就可以覆盖整段河道。

输入描述

输入包括 n+1n+1n+1 行。

第一行包括两个正整数 nnn 和 L (1≤n≤105, 1≤L≤109)L \ \left( 1 \leq n \leq 10^5, \ 1 \leq L \leq 10^9 \right)L (1≤n≤105, 1≤L≤109)。

接下来的 nnn 行，每行两个正整数 xix_ixi​ 和 yi (0≤xi≤yi≤109)y_i \ \left( 0 \leq x_i \leq y_i \leq 10^9 \right)yi​ (0≤xi​≤yi​≤109)，表示第 iii 个真视守卫覆盖的区间。

输出描述

一个整数，表示最少需要的真视守卫数量，如果无解，输出 -1。

示例

输入

4 6
3 6
2 4
0 2
4 7

输出

3

题解

思路如下：

  • 将所有区间按左端点从小到大排序。

  • 从前往后依次枚举每个区间，在所有能够覆盖 start 的区间中，选择右端点最大的区间，然后将 start 更新成右端点的最大值。

代码如下：

#include <iostream>
#include <cstdio>
#include <algorithm>
#include <climits>

using namespace std;

const int N = 100010;

struct Range
{
    int l, r;
    bool operator< (const Range& t) const
    {
        return l < t.l;
    }
} range[N];

int main()
{
    int n, start = 0, end;
    scanf("%d%d", &n, &end);

    for (int i = 0; i < n; i++)
    {
        int l, r;
        scanf("%d%d", &l, &r);
        range[i] = {l, r};
    }

    // 将所有区间按左端点从小到大排序
    sort(range, range + n);

    int res = 0;
    bool success = false;

    for (int i = 0; i < n; i++)
    {
        // 从前往后依次枚举每个区间，在所有能够覆盖start的区间中，选择右端点最大的区间
        int j = i, maxr = INT_MIN;
        while (j < n && range[j].l <= start)
        {
            maxr = max(maxr, range[j].r);
            j++;
        }

        if (maxr < start) break;

        res++;

        if (maxr >= end)
        {
            success = true;
            break;
        }

        // 将start更新成右端点的最大值
        start = maxr;
        i = j - 1;
    }

    if (!success) res = -1;

    printf("%d\n", res);

    return 0;
}
