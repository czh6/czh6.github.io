腾讯2020校园招聘-后台：逛街

发表于2023-03-30|更新于2023-03-30|笔试真题腾讯
|字数总计:752|阅读时长:2分钟|阅读量:

题目链接：https://www.nowcoder.com/questionTerminal/35fac8d69f314e958a150c141894ef6a

题目描述

小Q在周末的时候和他的小伙伴来到大城市逛街，一条步行街上有很多高楼，共有 nnn 座高楼排成一行。

小Q从第一栋一直走到了最后一栋，小Q从来都没有见到这么多的楼，所以他想知道他在每栋楼的位置处能看到多少栋楼呢？（当前面的楼的高度大于等于后面的楼时，后面的楼将被挡住）

输入描述

输入第一行将包含一个数字 nnn，代表楼的栋数，接下来的一行将包含 nnn 个数字 wi (1≤i≤n)w_i \ \left( 1 \leq i \leq n \right)wi​ (1≤i≤n)，代表每一栋楼的高度。

1≤n≤1000001 \leq n \leq 1000001≤n≤100000；

1≤wi≤1000001 \leq w_i \leq 1000001≤wi​≤100000；

输出描述

输出一行，包含空格分割的 nnn 个数字 viv_ivi​，分别代表小Q在第 iii 栋楼时能看到的楼的数量。

示例

输入

6
5 3 8 3 2 5

输出

3 3 5 4 4 4

说明

当小Q处于位置 333 时，他可以向前看到位置 2,12, 12,1 处的楼，向后看到位置 4,64, 64,6 处的楼，加上第 333 栋楼，共可看到 555 栋楼。

当小Q处于位置 444 时，他可以向前看到位置 333 处的楼，向后看到位置 5,65, 65,6 处的楼，加上第 444 栋楼，共可看到 444 栋楼。

题解

从前往后扫描一遍数组，时刻保持从栈底到栈顶的元素是单调递减的，对于当前待入栈的元素 a[i]a[i]a[i] 来说，此时栈中的元素个数就是在第 iii 栋楼时向左能看到的楼的数量。

从后往前扫描一遍数组，时刻保持从栈底到栈顶的元素是单调递减的，对于当前待入栈的元素 a[i]a[i]a[i] 来说，此时栈中的元素个数就是在第 iii 栋楼时向右能看到的楼的数量。

#include <iostream>
#include <cstdio>
#include <stack>

using namespace std;

const int N = 100010;

int a[N], lcnt[N], rcnt[N];

int main()
{
    int n;
    scanf("%d", &n);

    for (int i = 0; i < n; i++)
    {
        scanf("%d", &a[i]);
    }

    stack<int> stk;

    for (int i = 0; i < n; i++)
    {
        // 当前元素a[i]待入栈，此时栈中的元素个数就是在第i栋楼时向左能看到的楼的数量
        lcnt[i] = stk.size();

        while (!stk.empty() && a[i] >= stk.top())
        {
            stk.pop();
        }

        stk.push(a[i]);
    }

    stack<int>().swap(stk);    // 将栈stk清空

    for (int i = n - 1; i >= 0; i--)
    {
        // 当前元素a[i]待入栈，此时栈中的元素个数就是在第i栋楼时向右能看到的楼的数量
        rcnt[i] = stk.size();

        while (!stk.empty() && a[i] >= stk.top())
        {
            stk.pop();
        }

        stk.push(a[i]);
    }

    for (int i = 0; i < n; i++)
    {
        printf("%d ", lcnt[i] + rcnt[i] + 1);
    }

    return 0;
}
