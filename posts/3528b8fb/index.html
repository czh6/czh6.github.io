腾讯2020校园招聘-后台：压缩算法

发表于2023-03-27|更新于2023-03-29|笔试真题腾讯
|字数总计:806|阅读时长:3分钟|阅读量:

题目链接：https://www.nowcoder.com/questionTerminal/c27561e5b7e0441493adb9a54071888d

题目描述

小Q想要给他的朋友发送一个神秘字符串，但是他发现字符串过于长了，于是小Q发明了一种压缩算法对字符串中重复的部分进行了压缩，对于字符串中连续的 m 个相同字符串 S 将会压缩为 [m|S]，其中 mmm 为一个整数且 1≤m≤1001 \leq m \leq 1001≤m≤100，例如字符串 ABCABCABC 将会被压缩为 [3|ABC]，现在小Q的同学收到了小Q发送过来的字符串，你能帮助他进行解压缩么？

输入描述

输入第一行包含一个字符串 sss，代表压缩后的字符串。

sss 的长度小于等于 100010001000；

sss 仅包含数字、大写字母、[、]、|；

解压后的字符串长度不超过 100000100000100000；

压缩递归层数不超过 101010 层；

输出描述

输出一个字符串，代表解压后的字符串。

示例

输入

HG[3|B[2|CA]]F

输出

HGBCACABCACABCACAF

说明

HG[3|B[2|CA]]F −> HG[3|BCACA]F −> HGBCACABCACABCACAF

题解

本题可以找到原型：LeetCode 394. 字符串解码

本题既可以借鉴 LeetCode 20. 有效的括号 的思路利用栈来模拟字符串重复，也可以利用循环或递归反复替换重复的字符串。然而，如何做到高效呢？

我们为每层括号（即每个重复的部分）记录如下两个变量：

  • repeat_idx：记录重复部分的起始索引

  • repeat_cnt：记录重复部分的重复次数

有多层括号嵌套，我们用栈来保存每层的这两个变量信息。

从前往后遍历输入串 input：

  • 遇到大写字母：放进输出串 output

  • 遇到数字：统计重复的次数 repeat_cnt

  • 遇到 |：将确定好的两个变量信息入栈

  • 遇到 ]：出栈，同时将输出串 output 中的从 repeat_idx 开始的字符串复制 repeat_cnt - 1 次

整体上只扫描了一次输入串；输出串也是一直往后拼接的，没有回退。总时间复杂度和空间复杂度都是 O(n+m)O \left( n + m \right)O(n+m)，其中 nnn 为输入串长度，mmm 为输出串长度，与压缩递归层数无关。

#include <iostream>
#include <string>
#include <stack>
#include <utility>

using namespace std;

int main()
{
    string input, output;
    cin >> input;

    int repeat_idx = 0;    // 记录重复部分的起始索引
    int repeat_cnt = 0;    // 记录重复部分的重复次数
    stack<pair<int, int>> stk;

    for (auto &c : input)
    {
        if (c >= 'A' && c <= 'Z')
        {
            output += c;
        }
        else if (c == '[')
        {
            repeat_cnt = 0;
        }
        else if (c >= '0' && c <= '9')
        {
            repeat_cnt = repeat_cnt * 10 + c - '0'; // 重复部分的重复次数
        }
        else if (c == '|')
        {
            repeat_idx = output.size(); // 重复部分的起始索引
            stk.emplace(repeat_idx, repeat_cnt);
        }
        else if (c == ']')
        {
            auto [idx, cnt] = stk.top();
            stk.pop();
            auto tmp = output.substr(idx);
            for (int i = 0; i < cnt - 1; i++)
            {
                output += tmp;
            }
        }
    }

    cout << output << endl;

    return 0;
}
